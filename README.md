# shaedy AutoTeam

A CounterStrikeSharp plugin that automatically assigns connecting players to a team and forces respawn.

## Features

- Puts new players on a random team (T or CT) on connect
- Forces a respawn so players don't end up dead on join

## Installation

Drop the plugin folder into your CounterStrikeSharp `plugins` directory.

## Configuration

No config needed. Fully automatic.
